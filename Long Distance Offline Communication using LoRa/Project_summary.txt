## 🎯 Core Features

### **Transmitter (TX)**
- Reads GPS coordinates from module
- Checks for stale GPS data (rejects if >2s old)
- Adds sequence number to each packet
- Transmits via LoRa every 2 seconds
- Warns when commas removed from messages
- Shows GPS age in Serial Monitor

### **Receiver (RX)**
- Receives LoRa packets
- Displays on OLED screen
- Forwards to computer via USB serial
- Handles missing OLED gracefully
- Detects sequence numbers automatically
- Measures and reports RSSI

### **Web Dashboard**
- Live map with OpenStreetMap tiles
- Real-time GPS tracking
- Turn-by-turn road route (via OSRM)
- Straight-line distance calculation
- GPS track history (breadcrumb trail)
- Signal strength (RSSI) monitoring
- Packet sequence and drop detection
- Offline tile fallback
- Responsive design

## 🔧 Configuration

### **Receiver Location** (Required)
Edit `server.py` lines 42-44:
```python
RECEIVER_LAT = 19.076090  # Change this
RECEIVER_LNG = 72.877426  # Change this
```

### **Serial Port** (Optional)
Auto-detected, but can override:
```bash
# Windows
set LORA_PORT=COM5

# Mac/Linux  
export LORA_PORT=/dev/ttyUSB0
```

### **Network Access** (Optional)
Default = localhost only. For LAN access:
```bash
# Windows
set FLASK_HOST=0.0.0.0

# Mac/Linux
export FLASK_HOST=0.0.0.0
```

### **Custom Message**
In Serial Monitor (9600 baud):
```
Type message and press Enter
Example: Vehicle 01 - Moving
```

## 🛡️ Error Handling

### **Graceful Degradation**
- No serial port → waits and retries every 10s
- No internet → shows warning, markers still work
- No OLED → continues, retries every 30s
- Stale GPS → rejects, waits for fresh data
- Invalid packets → skips, logs error

### **User Feedback**
- Clear error messages
- Yellow warning banners
- Serial Monitor status updates
- Automatic reconnection
- No silent failures

→ Set LORA_PORT manually

## 🎓 How It Works

### **Data Flow**
```
TX: GPS → Arduino → LoRa Radio
         ↓ (wireless)
RX: LoRa Radio → Arduino → OLED Display
                          → USB Serial
                          ↓
PC: Python Server → Flask Web → Browser Dashboard
```

### **Packet Format**
```
Over LoRa: 0,19.076090,72.877426,Hello Trainee!
                ↓
Over USB:  DATA:0,19.076090,72.877426,Hello Trainee!,RSSI:-87
                ↓
Python:    {seq:0, lat:19.076090, lng:72.877426, msg:"Hello Trainee!", rssi:-87}
                ↓
JSON API:  Browser fetches /data every 3 seconds
                ↓
Dashboard: Map updates with new position
```
